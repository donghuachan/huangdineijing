# 《黄帝内经》与《易经》的关联
* 《易经》为外易，《黄帝内经》为内易，
```
《易经》讲天地宇宙外在变化的大规律，
《黄帝内经》讲人体生命内在变化的大规律。
```

# 《黄帝内经》现实意义
* 健康美丽，不在外求，在内求
* 治病最主要的方法其实不是吃药；“治未病”比“治已病”更重要。

# 内求
我认为《内经》还隐藏一个秘密：那就是对待生命的方法，一定要“内求”。

# 两部分
* 《素问》：对生命本质本原的追问。主要讲人体生命的基本理论问题。
* 《灵枢》原来称为《针经》，主要是讲经络和针灸的。
```
灵枢，意思就是主宰生命的枢纽和关键，
也就是神气、灵气运行的通道，
这个通道就叫经络，经络是生命的枢纽、神气的关键。
```

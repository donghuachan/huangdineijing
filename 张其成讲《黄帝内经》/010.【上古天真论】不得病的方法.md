```
“夫上古圣人之教下也，皆谓之虚邪贼风，避之有时，
恬淡虚无，真气从之，精神内守，病安从来？”
```

* 回避虚邪贼风： 一切致病的邪气，es. 六淫邪气（风、寒、暑、湿、燥、火）
* 内守：
  * 恬淡虚无： 少欲望，不追求命令
  * 
  * 精神内守：守住精气、神气，不外泄。真气顺从（任督二脉+十二经脉）而行
  
```
“是以志闲而少欲，心安而不惧，形劳而不倦，
气从以顺，各从其欲，皆得所愿。
故美其食，任其服，乐其俗，高下不相慕，其民故曰朴。”
```

* 心安： 把情志、思想挡住，控制住，不随意发展。心安才能无所畏惧
  * 当你领悟到心本虚空，不安的心也安了
* 回到最原始、淳朴、本真的状态

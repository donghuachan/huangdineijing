# “昔在黄帝，生而神灵，弱而能言，幼而徇齐，长而敦敏，成而登天。”
`黄帝的五个阶段，也是每个人一生的五个阶段，`
`人从生到死，就是神气从兴盛到衰亡的过程`
1. 生而神灵： 每个人生下来都神气十足
2. 弱而能言
```
小孩子问哲学问题：从哪来，去何处，我是谁
```
3. 幼而徇齐
```
幼小的时候行动迅速、快捷。
第一个阶段是刚生下来，
第二个阶段是会说话，
第三个阶段是会做事。
```

```
人生前三阶段：天真
天真就是我们婴幼儿、童年的那种天然、率真，那种单纯、质朴。

```

4. 长而敦敏
```
黄帝是“长而敦敏”，字面意思是长大了敦厚、敏锐，
实际意思是长大了还能保持儿时的天真，

```

5.成而登天
```
成年登上天子之位。(二十岁)
但这个”天“不是天子，而是”天年“，一个是时间的天，一个是空间的天
```

# 总结：
人一生的五个阶段
天真
